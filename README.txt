1. Importar el proyecto a Eclipse para su correcta ejecución.
2. al desplegar el proyecto ejecutelo como “Run on Server” y seleccione el servidor WildFly 10.x.
3. cuando el proyecto este en ejecución realiza las pruebas pertinentes, por ejemplo en Postman.
4. Las ruta de acceso a los servicios es “http://localhost:8080/RotondAndes/rest” .
5. para realizar alguna prueba podría intentar acceder a la lista de restaurantes desde el administrador “http://localhost:8080/RotondAndes/rest/administradores/1/restaurantes”.
